Unless/IF

If refers to a positive condition:
If I win I'll be happy.

Unless refers to a negative condition and can be used instead of if not when we refer to exceptional circumstances whish would change a situation:
Unless I win / If I don't win I won't be happy.

Unless ... = If not ...
